# MCP Servers Collection

This directory contains the source code for various specialized **Model Context Protocol (MCP)** servers. Each subdirectory is a standalone Go module or part of the hub workspace.

## Servers in this directory:

- **d2mcp**: Renders D2 diagrams and integrates with `mlcartifact`.
- **memory-server**: Provides a persistent knowledge graph for AI context.
- **openstreetmap_mcp**: Interface for fetching real-world geographic data.
- **task-manager**: A tool for managing and orchestrating complex AI tasks.

For build instructions, please refer to the [Root README](../README.md).
